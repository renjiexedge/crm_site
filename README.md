# Crm Site

This project is a dynamic website that acts as a crm_site to collect information of customer's relationships and store it in supabase.

## Project Structure

```
static-site
├── index.html       # Initial Login Page
├── script.js        # JavaScript file for interactivity
├── style.css        # CSS file for styling
└── homecare_crm.html   #Main HTML document
```

## Getting Started

To view the site, open the `index.html` file in a web browser. 

## Features

- Basic HTML structure
- Responsive design using CSS
- Interactive elements powered by JavaScript

## Installation

No installation is required. Simply clone the repository and open `index.html` in your preferred web browser.
